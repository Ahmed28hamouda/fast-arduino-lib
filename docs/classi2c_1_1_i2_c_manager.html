Detailed Description

template<I2CMode MODE_ = I2CMode::STANDARD>
class i2c::I2CManager< MODE_ >

General I2C Manager.

It is used by all I2C devices for transmission. For the time being, the MCU must always act as the only master on the bus. Using MCU as a slave will be supported in a later version of FastArduino.

Template Parameters
