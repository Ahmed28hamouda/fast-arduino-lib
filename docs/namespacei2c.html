Detailed Description

Define API to define and manage I2C devices.

I2C is available to all MCU supported by FastArduino, even in ATtiny MCU, for which I2C is implemented with Universal Serial Interface (USI).

Note
Current implementation supports only synchronous operation. Future FastArduino versions may also support asynchronous (non-blocking) operation.

The following snippet shows how to use an I2C device, the DS1307 Real Time Clock:

int main()
{
i2c::I2CManager<i2c::I2CMode::STANDARD> manager;
manager.begin();
devices::rtc::DS1307 rtc{manager};
devices::rtc::tm now;
rtc.get_datetime(now);
...
manager.end();
}

◆ BusConditions

enum i2c::BusConditions : uint8_t
strong

Bus condition (see I2C) to be generated by I2CDevice read and write methods.

This allows actual I2C devices to perform several reads and writes without "losing" the bus.

Enumerator
